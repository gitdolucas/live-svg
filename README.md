# Live SVG – Signature animator

![Live SVG](public/this-is-live-svg.svg)

Draw your signature, set timing and easing per stroke, then export an animated SVG.

Built with [Next.js](https://nextjs.org), React, and Tailwind CSS.

## Features

- **Draw** – Freehand drawing with optional path smoothing, color and stroke width
- **Timeline** – Per-stroke keyframes: delay, duration, and easing (linear, ease-in, ease-out, etc.)
- **Preview** – Live playback with configurable background
- **Export** – Download as a single animated SVG file

## Getting started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command   | Description              |
| --------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Build for production     |
| `npm run start` | Start production server  |
| `npm run lint`  | Run ESLint               |
